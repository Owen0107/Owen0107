### 💫 Về Mình
🚀 Chào bạn! Mình là một lập trình viên đam mê xây dựng các ứng dụng web tương tác trực quan, tối ưu hóa hệ thống dữ liệu và đặc biệt yêu thích công nghệ **3D Web Integration (Digital Twin)**.

- 🏢 **Hiện tại:** Đang thực tập và học hỏi các kiến thức thực tế về phát triển phần mềm doanh nghiệp.
- 🛠️ **Dự án cốt lõi:** Tập trung phát triển hệ thống quản lý kho thông minh tích hợp mô hình 3D.
- 🎮 **Giải trí:** Hardcore gaming (PUBG, Grounded) và nghiên cứu các công cụ/macro tối ưu hiệu năng trò chơi bằng code, có CountMeterPUBG do mình tạo ra để tính toán bắn cối.
